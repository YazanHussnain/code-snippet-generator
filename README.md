# Email Code Snippet Generator

This tool generates **email-safe HTML code snippets**
with **line numbers**, **syntax highlighting**, and
**bold reference lines**.

## How to Use

1. Open `index.html` in your browser
2. Paste your C/C++ or Python code
3. Select language
4. Enter lines to emphasize:
   - `3`
   - `5,7,12`
   - `10-14`
5. Click **Generate**
6. Click **Copy HTML**
7. Paste directly into Gmail / Outlook email body

## Why This Exists

- Email clients do not support Markdown or JS
- Reviewers want precise line references
- Ideal for design reviews, code walkthroughs,
  and spec discussions

## Supported
- C / C++
- Python

## Tested With
- Gmail (Web)
- Outlook (Desktop & Web)
